## üöÄ **Getting Started**

### **Installation**
```bash
cd kgpgpt
npm install
npm run dev
```

### **Environment Setup**
Create a `.env.local` file with:
```env
GEMINI_API_KEY=your_gemini_api_key
SERPER_API_KEY=your_serper_api_key
QDRANT_URL=http://localhost:6333
```

### **Features Usage**

#### **Quick Start Templates**
- Click on any template in the sidebar to start a conversation
- Templates are categorized by use case

#### **Message Reactions**
- Hover over any message to see action buttons
- Click the three dots to add reactions
- Use thumbs up/down for feedback

#### **File Uploads**
- Click the paperclip icon to attach files
- Drag and drop files directly
- Supports multiple file types

#### **System Monitoring**
- Click on the system status indicator for details
- Monitor real-time performance metrics
- Track system health and connections
